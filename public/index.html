Destiny 2 Data Connector

Connect to Bungie's Destiny 2 API and analyze Guardian data in Tableau

About This Connector

This Web Data Connector allows you to visualize Destiny 2 weapon usage statistics within Tableau. Access detailed metrics such as kill counts, precision kills, and usage time to analyze Guardian weapon preferences and effectiveness across both PvE and PvP activities.

Weapon Usage Data

This connector will return the following weapon statistics:

Weapon Name:
Names of Destiny 2 weapons used by Guardians
Weapon Type:
The category of each weapon (Hand Cannon, Auto Rifle, etc.)
Kill Count:
Total number of enemies defeated with this weapon
Precision Kills:
Number of precision kills (headshots) achieved with this weapon
Usage Time:
Total time spent using this weapon (in seconds)
Activity Type:
Whether the weapon was used in PvE or PvP activities
Date:
The date for which the data was collected (optional)
